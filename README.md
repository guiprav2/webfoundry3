# Webfoundry

Webfoundry is an innovative site builder designed to streamline the process of creating and managing websites. With a user-friendly interface and a range of powerful features, Webfoundry is ideal for both beginners and experienced developers.

## Features

- **Sidebar for Easy Management**: Manage your sites, files, and CSS class names effortlessly with the intuitive sidebar.
- **ZIP Import/Export**: Easily import and export your site data with the convenient ZIP file functionality.
- **CSS Class Name Management**: Directly manage CSS class names for selected elements, enhancing your ability to customize and style your website.
- **User-Friendly Interface**: Designed with simplicity and efficiency in mind, making website management accessible to everyone.
- **Command Help**: Click the question mark icon in the sidebar header for a list of commands.

## Getting Started

To get started with Webfoundry, follow these steps:

1. **Clone the Repository**:

```sh
$ git clone https://github.com/guiprav2/webfoundry3.git
$ cd webfoundry3
```

2. **Serve with npx serve**:

```sh
$ npx serve
```

3. **Go to https://localhost:3000/**

Go to https://localhost:3000/ to use the app.
